How to use the Kqurada_Cash Book.

The notes in the book represents real money. Everytime you sell a product or service that it's price corresponds with the amount in the book, then scan the amount in the book which will be automatically listed in the scanned list section including the total.

Incase you sell something that doesn't have a corresponding price in the book, then just scan any higher amount then record the extra amount in the discount section, after which you'll subtract all the extras recorded plus all the recorded expenses to get your day's net sales amount.


PDF Section Explained

1. Name Input

This is where you type your name (the person taking the records) or that of the business

2. Date

This is where real time is displyed.

3. Scanned Amount List with Total.

This is where all scanned sales list with total is displayed. It has two elements:

  • Subtract Last Button: deletes and subtracts last scanned amount.
  • Delete All Button: Deletes all the scanned sales.

4. Stock Records

Has three sections as follows:

(A). Add Stock

Has three elements as follows:

  • Stock Item Input: This is where you type the name of item that you are adding to stock.
  • Stock Quantity Input: Type the quantity of item that you are stocking here.
  • Add Stock Button: Click this button to add the item to stock table.

(B). Sell Stock

Has three elements as follows:

  • Sell Item: Choose item to sell from the drop down menu where every added item is placed.
  • Quantity Input: Type the quantity you are selling here i.e 1, 2, 3, 4....etc.
  • Sell Button: Click this button to sell item.

(C). Stock Table

This is where all stock records are displayed. It has 4 columns:

  • Column 1: Item Name - name of stocked item.
  • Column 2: Sold Count - shows quantity sold for each item.
  • Column 3: Remaining Stock - shows remaining quantity of each item after selling.
  • Column 4: Action - Cointains delete buttons for each item.
  • Clear All Button - For deleting everything in the table.

5.Sold Services

This is where you record all the servies sold. It has three elements:

  • Name Input: type the name of sold service here.
  • Add Item button: click this button to add service item to the list.
  • Clear All Button: Deletes all the added items.

6. Expenses and Discounts

Record all expenses amd discounts here. Discount here mean the extra amount scanned. See how to use the book above to know more. It has five elements as follows;

  • Expense Name Input
  • Expense Amount Input
  • Add Expense Button
  • Display for Total and
  • Clear All Button

7.Subtract Expenses and Discounts

This is where you subtract all the expenses smd discounts to get the total net sale(s) of the day.

N/B See how to use the Kqurada_Cash book to know what discount mean here.

8. Save Content Button

This button saves the records to your device as either pdf or image for future reference. Pdf recomended.
